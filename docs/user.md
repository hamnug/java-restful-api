# User API Spec

## Register User

Endpoint : POST /api/users/register

Request Body : 
```json
{
    "username" : "hamnug",
    "password" : "rahasia",
    "name" : "Muhammad Ilham Nugraha"
}
```

Response Body (Success) : 
```json
{
    "data" : "OK"
}
```

Response Body (Failed) :
```json
{
    "error" : "Incorrect Username or Password"
}
```
